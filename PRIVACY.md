# 隐私政策

欢迎使用 AI-ViewNote（以下简称"本软件"）。本隐私政策说明了我们如何收集、使用、存储和保护您的信息。

## 1. 信息收集

### 1.1 本地处理
- 本软件是一款本地应用程序，所有视频处理和 AI 分析都在您的设备本地进行
- 我们不会自动收集、传输或存储您的个人数据到外部服务器

### 1.2 用户上传内容
- 您上传到本软件的视频文件仅在您的设备本地处理
- 这些文件不会被传输到我们的服务器或第三方服务（除非您主动配置外部 AI 服务）

## 2. 信息使用

### 2.1 本地处理目的
- 视频文件：用于提取音频和生成转录文本
- 音频文件：用于语音识别和文字转录
- 生成的内容：用于创建结构化的 Markdown 笔记

### 2.2 不收集的信息
我们不会收集以下信息：
- 个人身份信息（姓名、邮箱、电话号码等）
- 设备标识符
- 使用行为数据
- 位置信息

## 3. 第三方服务

### 3.1 外部 AI 服务
- 本软件支持配置外部 AI 服务（如 OpenAI、火山引擎 等）
- 如果您选择使用这些服务，您的音频数据可能会发送到相应的服务提供商
- 这些服务的使用受其各自的隐私政策约束

### 3.2 GitHub 服务
- 检查更新功能会连接到 GitHub API 获取最新版本信息
- 这可能会向 GitHub 发送基本的请求信息

## 4. 数据存储

### 4.1 本地存储
- 所有处理的数据都存储在您的设备本地
- 您可以随时删除本地数据
- 本软件不会在云端存储您的任何数据

### 4.2 临时文件
- 处理过程中可能生成临时文件，这些文件会在处理完成后自动清理

## 5. 数据安全

### 5.1 本地保护
- 由于所有数据都在本地处理，数据安全主要依赖于您的设备安全措施
- 建议您保护好自己的设备，避免未授权访问

### 5.2 传输安全
- 仅在您主动配置外部服务时才会发生数据传输
- 建议使用安全的网络连接

## 6. 儿童隐私

本软件不针对 13 岁以下儿童设计，我们不会有意收集儿童个人信息。

## 7. 隐私政策更新

我们可能会不时更新本隐私政策。重大变更时，我们会在 GitHub 仓库中更新政策内容。

## 8. 您的权利

### 8.1 访问权
- 您可以随时查看本地存储的所有数据

### 8.2 删除权
- 您可以随时删除本地存储的所有数据

### 8.3 撤回同意
- 您可以随时停止使用本软件

## 9. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过 GitHub Issues 联系我们：
[https://github.com/liangbm3/AI-ViewNote/issues](https://github.com/liangbm3/AI-ViewNote/issues)

## 10. 适用法律

本隐私政策受中华人民共和国法律管辖并按其解释。
